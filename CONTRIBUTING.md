# Contributing to Payload Extractor GUI

感谢你对这个项目的关注！

## 关于本项目

这是一个个人自用的小工具，主要在 macOS 上使用。代码由 Claude AI 协助完成。

虽然这是个人项目，但如果你有好的想法或发现了 bug，欢迎：

- 🐛 提交 Issue 报告问题
- 💡 分享改进建议
- 🔧 Fork 后自行修改使用

## 测试环境

- **系统**: macOS (主要测试环境)
- **Python**: 3.8+
- **payload-dumper-go**: 最新版本

## 简单贡献指南

如果你想改进这个工具：

1. Fork 本仓库
2. 创建你的分支: `git checkout -b feature/your-feature`
3. 提交更改: `git commit -m "Add: your feature"`
4. 推送: `git push origin feature/your-feature`
5. 提交 Pull Request

## 代码风格

- 遵循 PEP 8 规范
- 保持代码简洁易读
- 添加必要的注释

## 许可

贡献的代码将采用 MIT 许可证。
